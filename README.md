# HotelPrices

Projeto MVC de WebScraper que contém:

+ Consulta preços de um Hotel através de WebScraper

# Tecnologias

+ . C#
+ . .NET 6
+ . EntityFramework
+ . PostgreSQL (RailWay)

# Passo a passo de uso

+ Rode o projeto;
+ Coloque uma data de CheckIn
+ Coloque uma data de ChckOut
+ Coloque a quantidade de Adultos
+ Clique no botão Pesquisar

# Sobre

+ Projeto seguindo padrão MVC, princípios SOLID, boas práticas e legibilidade.
+ Banco de dados PostgreSQL na nuvem com RailWay.
